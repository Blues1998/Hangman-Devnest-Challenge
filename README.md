# Hangman-Devnest-Challenge

<h2>
  Solution for HangMan challenge
</h2>

<h3>
  Usage:
</h3>

<p>
    1. Clone this repository in your system with 
    <pre>
    <code>
      git clone https://github.com/Blues1998/Hangman-Devnest-Challenge.git
      cd Hangman-Devnest-Challenge/
    </code>
    </pre>
    2. Run using Python3
    <pre>
    <code>
      python hangman.py
    </code>
    </pre>
</p>
